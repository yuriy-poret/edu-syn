# FinTech Wallet API

## Описание проекта

FinTech Wallet API — это backend-приложение для управления персональными финансовыми операциями.
Система позволяет пользователю:
- зарегистрироваться и авторизоваться;
- добавлять доходы и расходы;
- просматривать, редактировать и удалять транзакции;
- получать финансовую аналитику;
- вести журнал действий пользователя.

## Примеры API

### Регистрация
POST /auth/register

### Авторизация
POST /auth/login

### Получение текущего пользователя
GET /users/me

### Создание транзакции
POST /transactions

### Получение списка транзакций
GET /transactions

### Получение аналитики
GET /analytics/summary

### Получение журнала действий
GET /audit-logs
